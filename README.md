# Academic Subjects Portal

This is a simple and organized web project that displays subjects year-wise for students. Each subject leads to its own page containing study materials (like PDFs) categorized by academic year.

## 🌟 Features
- Clean UI with animated subject cards
- Year-wise and subject-wise navigation
- Organized folder structure for scalability

## 🛠️ Tech Stack
- HTML
- CSS

## 📁 Folder Structure
- `/Main` - Contains the parent HTML and CSS
- `/Subjects` - Contains all subject-specific folders and materials
- `/Years` - Contains year-specific HTML files
